# 🚀 SilentShare

SilentShare is an **offline messaging application** that enables communication between devices **without internet** using **WiFi Direct**.

---

## 📸 Screenshots

> Add your screenshots in a `screenshots` folder



## ✨ Features

- 📡 Offline messaging (No Internet Required)
- 🔗 WiFi Direct device-to-device connection
- 👤 Profile creation (Name + Avatar)
- 💬 Real-time messaging
- ⚡ Fast connection & communication
- 🔐 Secure communication channel

### Chat Modes
- Individual Chat
- Group Chat (Coming Soon)
- Broadcast Chat (Coming Soon)

---

## 🧠 Tech Stack

- Kotlin
- Jetpack Compose
- WiFi Direct (P2P)
- Coil (Image loading)
